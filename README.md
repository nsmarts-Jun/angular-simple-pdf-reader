# angular-simple-pdf-reader


## 개요

웹 클라이언트 프레임워크인 Angular를 사용해, 국제 표준 전자 문서 형식인 PDF를 읽기, 저장, 공유 가능한 웹 서비스를 개발.

## 환경설정

Angular  12.2.9 

node    v14.17.3

npm      6.14.13


## 실행

```
ng serve
```
